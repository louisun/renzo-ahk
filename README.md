# renzo-ahk

- 按键映射
- 唤起、隐藏窗口，优先唤起同一应用最近使用的窗口
- 切换全局窗口
- 切换同一应用下的多个窗口

## 窗口切换

- 上下移动：w / s / j / k 

- 激活窗口：双击 / space / d / l
- 退出：q / 鼠标点击其他窗口
